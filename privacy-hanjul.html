한줄일기 개인정보 처리방침

최종 업데이트: 2026년 4월 23일

개요

한줄일기(Hanjul)는 사용자가 하루 한 줄의 일기를 남기면 AI 가 따뜻한 답글을 보내주는 앱입니다. 서비스를 제공하기 위해 필요한 최소한의 데이터만을 처리하며, 개인 식별 정보(이름·이메일·전화번호·로그인 계정 등)는 일체 수집하지 않습니다.

수집·처리하는 정보

한줄일기는 다음 정보를 처리합니다:

  • 사용자가 직접 입력한 일기 텍스트 — 한 항목당 최대 100자. AI 답글 생성을 위해 일시적으로 서버를 거쳐 OpenAI 에 전달됩니다(아래 "데이터 흐름" 참고).
  • 익명 기기 식별자 — 앱이 최초 실행될 때 무작위 문자열을 생성해 기기 내부에만 저장합니다. 사용량 제한(rate limit) 에만 사용되며, 개인 식별 불가능합니다. 앱을 삭제하면 함께 사라집니다.

이름·이메일·전화번호·위치·광고 식별자·연락처·사진·마이크 접근 권한 등은 일체 수집하지 않으며, 앱은 회원가입·로그인 절차가 없습니다.

데이터 흐름 (AI 답글)

사용자가 일기를 작성하고 "답글 받기" 버튼을 누르면 다음 순서로 처리됩니다:

  • ① 앱 → Cloudflare Workers 프록시(hanjul-proxy.ssamssae.workers.dev): 일기 텍스트와 익명 기기 식별자가 HTTPS 로 전송됩니다.
  • ② 프록시 → OpenAI API(GPT-4o-mini): 일기 텍스트와 답글 생성용 시스템 프롬프트만 전달됩니다. 기기 식별자는 전달되지 않습니다.
  • ③ OpenAI 가 생성한 답글이 프록시를 거쳐 앱으로 돌아와 화면에 표시됩니다.

프록시는 일기 내용을 저장하지 않습니다. 기기 식별자 기준 요청 카운트(rate limit) 만 일시적으로 보관합니다.

OpenAI 의 API 정책상 전달된 텍스트는 OpenAI 모델 학습에 사용되지 않으며, 남용 감지 목적으로 최대 30일간 보관될 수 있습니다. 자세한 사항은 OpenAI API Data Usage Policies 를 확인해 주세요.

데이터 저장

사용자가 작성한 일기와 AI 답글은 사용자 기기 로컬 저장소(Android: SharedPreferences, iOS: UserDefaults)에만 영구 보관됩니다. 한줄일기 앱 자체는 사용자별 데이터를 서버에 저장하지 않으며, 어떤 계정에도 동기화되지 않습니다. 앱을 삭제하면 기기 내부의 모든 일기·답글 데이터가 함께 삭제됩니다.

제3자 제공 및 공유

한줄일기는 광고·분석 SDK·크래시 리포팅 도구 등 어떠한 외부 추적 서비스도 포함하지 않습니다. 다만 AI 답글 생성을 위해 아래 서비스를 이용합니다:

  • Cloudflare, Inc. — 프록시 인프라 제공. 요청 전송 과정에서 표준 HTTP 로그(IP·요청 시각·응답 코드) 가 최대 30일간 보관될 수 있습니다.
  • OpenAI, L.L.C. — AI 답글 생성 모델(GPT-4o-mini) 제공. 위 "데이터 흐름" 항목 참고.

이 두 서비스 외에는 어느 제3자에게도 사용자의 데이터를 제공하거나 공유하지 않습니다.

사용량 제한 및 예산

서비스의 지속 가능한 운영과 남용 방지를 위해 다음 제한을 두고 있습니다:

  • 기기당 하루 일정 횟수 제한 및 IP 당 하루 60회 제한
  • 월간 API 호출 예산 초과 시 일시적 이용 제한

제한 초과 시 앱 내부에서 "오늘 답글을 많이 받으셨어요" 또는 "잠깐 AI 가 쉬는 중" 과 같은 안내 문구가 표시됩니다. 일기 자체의 저장은 제한 여부와 무관하게 정상적으로 이뤄집니다.

어린이 개인정보 보호

한줄일기는 만 13세 미만 어린이를 대상으로 하지 않습니다. 만 13세 미만 이용자의 개인정보를 의도적으로 수집하지 않으며, 어린이임이 확인되는 경우 해당 데이터를 즉시 삭제합니다.

보안

앱과 프록시 사이의 모든 통신은 HTTPS(TLS) 로 암호화됩니다. 기기 내부 저장소는 운영체제의 앱 샌드박싱으로 보호되며, 민감한 내용을 기록하는 경우 기기 자체의 잠금 기능을 활성화하는 것을 권장합니다.

사용자 권리

한줄일기는 서버에 사용자별 데이터를 보관하지 않으므로 별도의 계정 삭제 절차가 없습니다. 앱 내부에서 개별 일기 항목을 삭제하거나 앱 자체를 기기에서 삭제함으로써 모든 데이터를 완전히 제거할 수 있습니다.

정책 변경

이 개인정보 처리방침은 앱의 기능 변경이나 제3자 서비스 정책 변경에 따라 업데이트될 수 있습니다. 변경 사항이 있을 경우 이 페이지에 반영되며, 최종 업데이트 날짜가 갱신됩니다.

메모요 개인정보 처리방침

최종 업데이트: 2026년 5월 13일

개요

메모요(Memoyo)는 꼭 필요한 것만 남긴 심플한 다크 모드 메모 앱입니다. 회원가입·로그인·서버 동기화·광고·분석 SDK 가 없으며, 어떤 개인 식별 정보(이름·이메일·전화번호 등) 도 수집하지 않습니다.

수집·처리하는 정보

메모요는 어떠한 개인정보도 수집하거나 외부로 전송하지 않습니다. 사용자가 앱에서 입력하는 정보는 다음과 같이 처리됩니다:

  • 메모 본문 — 사용자가 직접 작성한 메모 텍스트.
  • 즐겨찾기 표시 — 스와이프로 고정한 메모 식별 정보.
  • 메모 정렬 순서 — 드래그로 사용자가 직접 지정한 순서.

위 항목은 모두 사용자 기기 내부 로컬 저장소(iOS / Android 의 SharedPreferences) 에만 저장되며, 외부 서버나 제3자에게 전송되지 않습니다.

수집하지 않는 정보

  • 이름·이메일·전화번호·생년월일
  • 위치·연락처·사진·마이크·카메라 접근 권한
  • 광고 식별자(IDFA/AAID)·기기 식별자
  • 네트워크 사용 기록·크래시 로그

제3자 제공 및 공유

메모요는 광고·분석 SDK·크래시 리포팅 도구 등 어떠한 외부 추적 서비스도 포함하지 않으며, 어느 제3자에게도 사용자의 데이터를 제공하거나 공유하지 않습니다.

네트워크 통신

메모요 앱은 외부 서버와 통신하지 않습니다. 인터넷 연결 없이도 모든 기능이 정상 동작합니다.

기기 센서 사용

메모요는 인터랙션 보조를 위해 기기 모션 센서를 활용할 수 있으나, 센서 데이터는 모두 앱 내부에서만 처리되며 외부로 전송되거나 저장되지 않습니다.

어린이 개인정보 보호

메모요는 모든 연령대가 사용할 수 있는 앱이지만, 만 13세 미만 어린이의 개인정보를 수집하지 않습니다(애초에 어떠한 개인정보도 수집하지 않습니다).

사용자 권리 및 데이터 삭제

메모요는 서버에 사용자 데이터를 보관하지 않으므로 별도의 계정 삭제 절차가 없습니다. 앱을 기기에서 삭제하면 모든 데이터(메모 본문, 즐겨찾기, 정렬 순서) 가 함께 즉시 제거됩니다.

정책 변경

이 개인정보 처리방침은 앱의 기능 변경에 따라 업데이트될 수 있습니다. 변경 사항이 있을 경우 이 페이지에 반영되며, 최종 업데이트 날짜가 갱신됩니다.

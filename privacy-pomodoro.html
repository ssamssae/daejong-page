포모도로 개인정보 처리방침

최종 업데이트: 2026년 4월 23일

개요

포모도로(Pomodoro)는 25분 집중 / 5분 휴식 사이클을 가장 단순하게 굴리는 타이머 앱입니다. 서비스를 제공하기 위해 필요한 정보 이외에는 어떤 데이터도 수집·전송하지 않으며, 개인 식별 정보(이름·이메일·전화번호·로그인 계정 등)는 일체 수집하지 않습니다.

수집·처리하는 정보

포모도로는 사용자의 개인정보를 수집하지 않습니다. 앱이 다루는 모든 데이터는 사용자 기기 내부에서만 생성·저장되며 외부로 전송되지 않습니다.

기기 내부에 저장되는 항목:

  • 오늘 완료한 포모도로 사이클 수 — 화면 표시를 위한 카운터.
  • 최근 7일 간의 일별 집중 세션 수 — 주간 통계 그래프 표시를 위해 기기 로컬에 저장.
  • 타이머 상태 — 앱을 다시 실행했을 때 이어가기 위해 잠시 보관. 앱 종료 시 초기화될 수 있음.

이름·이메일·전화번호·위치·광고 식별자·연락처·사진·마이크·카메라 등 어떠한 개인정보도 수집하지 않으며, 앱은 회원가입·로그인 절차가 없습니다.

데이터 저장

위 정보는 사용자 기기 로컬 저장소(Android: SharedPreferences, iOS: UserDefaults)에만 보관됩니다. 포모도로 앱은 사용자별 데이터를 서버에 저장하지 않으며, 어떤 계정에도 동기화되지 않습니다. 앱을 삭제하면 기기 내부의 모든 포모도로 관련 데이터가 함께 삭제됩니다.

네트워크 통신

포모도로는 서비스 동작 과정에서 외부 서버와 통신하지 않습니다. 앱은 인터넷 연결 없이도 모든 기능이 정상 동작하며, 광고·분석·크래시 리포팅 SDK 등 어떠한 외부 네트워크 모듈도 포함하지 않습니다.

제3자 제공 및 공유

포모도로는 사용자의 데이터를 어떤 제3자에게도 제공·공유·판매하지 않습니다. 서비스 운영에 이용되는 외부 서비스(광고 네트워크, 분석 도구, 크래시 리포팅 등)가 없습니다.

권한(Permissions)

포모도로는 Android / iOS 에서 다음 권한 외에는 요청하지 않습니다:

  • 알림 표시 권한 — 25분 집중 / 5분 휴식 사이클 완료 시 로컬 알림을 띄우기 위해 사용. 원격 푸시 서버는 사용하지 않음.

위 권한은 사용자가 운영체제 설정에서 언제든 비활성화할 수 있습니다. 권한을 비활성화해도 타이머 자체의 기본 동작에는 영향이 없습니다.

어린이 개인정보 보호

포모도로는 만 3세 이상 모든 연령이 사용할 수 있는 생산성 도구이며, 아동을 대상으로 식별 정보를 수집하지 않습니다. 서비스 전반이 계정·로그인·네트워크 전송 없이 동작하므로 어린이로부터 수집되는 개인정보 자체가 존재하지 않습니다.

보안

기기 내부 저장소는 운영체제의 앱 샌드박싱으로 보호됩니다. 포모도로가 별도로 전송하는 네트워크 트래픽이 없으므로 중간 전송 구간의 보안 이슈도 발생하지 않습니다.

사용자 권리

포모도로는 서버에 사용자별 데이터를 보관하지 않으므로 별도의 계정 삭제 절차가 없습니다. 앱 자체를 기기에서 삭제함으로써 모든 데이터를 완전히 제거할 수 있습니다.

정책 변경

이 개인정보 처리방침은 앱의 기능 변경이나 법규 변경에 따라 업데이트될 수 있습니다. 변경 사항이 있을 경우 이 페이지에 반영되며, 최종 업데이트 날짜가 갱신됩니다.

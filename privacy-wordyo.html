단어요 개인정보 처리방침

최종 업데이트: 2026년 5월 2일

개요

단어요(Wordyo)는 영어 단어를 카드 형태로 학습하는 앱입니다. 서비스를 제공하기 위한 모든 처리는 사용자 기기 안에서만 이뤄지며, 외부 서버나 제3자 API 로 어떤 데이터도 전송하지 않습니다. 개인 식별 정보(이름·이메일·전화번호·로그인 계정 등)는 일체 수집하지 않습니다.

수집·처리하는 정보

단어요는 다음 정보를 사용자 기기 내부에서만 처리합니다:

  • 학습 진행도 — 어떤 카테고리의 어느 카드까지 봤는지를 기기 로컬 저장소에만 기록합니다.
  • 즐겨찾기·메모 등 사용자 입력 — 사용자가 카드에 직접 표시한 데이터는 기기 로컬에만 저장됩니다.

이름·이메일·전화번호·위치·광고 식별자·연락처·사진·마이크 접근 권한 등은 일체 수집하지 않으며, 앱은 회원가입·로그인 절차가 없습니다.

데이터 흐름 (외부 호출 없음)

단어요는 어떤 외부 서버, API, 클라우드 서비스와도 통신하지 않습니다. 단어 카드 데이터는 앱 설치 시 함께 포함되어 있으며, 학습 중 추가로 다운로드하거나 업로드하는 데이터가 없습니다. 인터넷 연결이 끊긴 상태에서도 모든 기능이 동일하게 작동합니다.

데이터 저장

학습 진행도, 즐겨찾기, 사용자 입력 등 모든 데이터는 사용자 기기 로컬 저장소(Android: SharedPreferences, iOS: UserDefaults)에만 보관됩니다. 단어요 앱 자체는 사용자별 데이터를 서버에 저장하지 않으며, 어떤 계정에도 동기화되지 않습니다. 앱을 삭제하면 기기 내부의 모든 학습 데이터가 함께 삭제됩니다.

제3자 제공 및 공유

단어요는 광고 SDK, 분석 SDK, 크래시 리포팅 도구, 외부 API 등 어떠한 외부 추적·전송 서비스도 포함하지 않습니다. 어느 제3자에게도 사용자의 데이터를 제공하거나 공유하지 않습니다.

어린이 개인정보 보호

단어요는 만 13세 미만 어린이를 대상으로 하지 않습니다. 만 13세 미만 이용자의 개인정보를 의도적으로 수집하지 않으며, 어린이임이 확인되는 경우 해당 데이터를 즉시 삭제합니다.

보안

단어요는 외부 통신을 수행하지 않으므로 네트워크 경로상의 데이터 노출 위험이 없습니다. 기기 내부 저장소는 운영체제의 앱 샌드박싱으로 보호되며, 민감한 메모를 기록하는 경우 기기 자체의 잠금 기능을 활성화하는 것을 권장합니다.

사용자 권리

단어요는 서버에 사용자별 데이터를 보관하지 않으므로 별도의 계정 삭제 절차가 없습니다. 앱 자체를 기기에서 삭제함으로써 모든 학습 데이터를 완전히 제거할 수 있습니다.

정책 변경

이 개인정보 처리방침은 앱의 기능 변경에 따라 업데이트될 수 있습니다. 변경 사항이 있을 경우 이 페이지에 반영되며, 최종 업데이트 날짜가 갱신됩니다.

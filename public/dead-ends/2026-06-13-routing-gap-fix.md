# 노드 라우팅 갭 픽스 — 재현 불가로 취소

- **드롭 시점:** 2026-06-13
- **분류:** 기술적으로 막힘
- **목표:** 노드 간 directive 라우팅이 가끔 queued로 멈추는 신뢰성 갭 수정

## 뭘 하려 했나

- 증상·원인·조치·픽스 A/B/C 초안까지 작성
- trio 검토에서 HOLD(2-1) — 재현·범위 불명확

## 왜 막혔나

- 갭을 안정적으로 재현하는 케이스를 못 잡음
- 재현 없이는 픽스가 맞는지 검증 불가 → 설계만 미정인 채 잔존

## 결론

재현 케이스 없이 진행 불가로 취소. 실제 재현 시 재오픈.

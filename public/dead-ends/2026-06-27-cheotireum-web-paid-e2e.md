# 첫이름 웹 유료결제 E2E 검증 — 앱 경로로 전환

- **드롭 시점:** 2026-06-27
- **분류:** 판단으로 접음
- **목표:** 첫이름 웹에서 실결제까지 도는 전 구간 E2E를 자동 검증

## 뭘 하려 했나

- 시뮬레이션 → 데이터 → 테스트 결제 → PDF 생성까지 풀 파이프 검증 위임 시도

## 왜 드롭했나

- 웹 라이브 유료 경로 자체를 더 끌고 가지 않기로 방향 변경
- 제품 무게중심을 웹이 아니라 모바일 앱으로 이동

## 결론

웹 유료 E2E는 접고 모바일 앱 E2E 수정으로 선회.

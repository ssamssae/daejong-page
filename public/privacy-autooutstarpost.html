처리 흐름

autooutstarpost 의 동작은 다음 한 줄로 끝납니다.

  • 렌더링된 이미지를 본인의 GitHub Pages(work.kangdaejong.com)에 호스팅한다.
  • Meta Graph API 를 호출해 본인의 인스타그램 비즈니스 계정에 캐러셀로 게시한다.

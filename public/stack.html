백엔드 구성

2026-06 기준 — 전 노드 메인 엔진은 Claude Code. Codex 는 다기기 합의 투표(mesh-vote) 전용으로 운용한다.

메인 엔진5노드 전부 Claude Code 동등 first-class. 야간 자동 사이클 = loop-fleet / 맥미니 오케스트레이터.
Codexcodex-mesh-vote 전용 — 5기기가 병렬로 브레인스토밍 후 상호 투표로 다수결. 일상 자동 워커는 Claude.
API fallbackClaude 토큰 소진 시 OpenRouter API 로 폴백. 직구 무료티어 우선, 비상 시 우회.
메시·스킬mesh-vote / loop-fleet / Agent Mesh Mirror 그룹으로 노드 간 명령·보고를 양방향 미러.

기기 연결 토폴로지

5노드 hub-and-spoke 메시 — 🍎 본진이 hub. 각 노드에 *-directive.sh 로 명령 운반(시안 실선) + mac-report.sh 회신. 노드 간 ssh 직결(점선)도 가능.

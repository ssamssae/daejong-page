## ASC keywords (100자 이내, 콤마 구분)

iOS App Store Connect 키워드 필드는 100자 제한 + 콤마 1개당 1자 차감. 키워드 자체엔 공백 안 넣는 게 정석 (제목·부제·설명 본문에서 이미 다회 매칭되기 때문에 키워드 필드는 압축).

```
메모,메모장,노트,노트앱,심플,미니멀,다크모드,즐겨찾기,스와이프,다이어리,일기,할일,드래그,정리,아이디어,광고없는,빠른메모,기록
```

길이 계산 (한글 1자 = 1자, 콤마 1자):
- 메모(2)+,(1)+메모장(3)+,(1)+노트(2)+,(1)+노트앱(3)+,(1)+심플(2)+,(1)+미니멀(3)+,(1)+다크모드(4)+,(1)+즐겨찾기(4)+,(1)+스와이프(4)+,(1)+다이어리(4)+,(1)+일기(2)+,(1)+할일(2)+,(1)+드래그(3)+,(1)+정리(2)+,(1)+아이디어(4)+,(1)+광고없는(4)+,(1)+빠른메모(4)+,(1)+기록(2)
- 누적: 3+4+3+4+3+4+5+5+5+5+3+3+4+3+5+5+5+2 = 76자

24자 여유. 추가 후보 — `심플메모(5), 미니멀메모(6), 노트필기(5)` 중 하나 더 가능. `심플메모,미니멀메모` 추가하면 76+5+6 = 87자 (콤마 2 포함 → 89자). 13자 여유. 최종안에 반영.

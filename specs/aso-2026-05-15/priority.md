## 적용 우선순위 — 4앱 순위 + 이유

### 순위: 약먹자 > 단어요 > 더치페이 > 메모요

#### 1위 — 약먹자 🏆

**예상 다운로드 임팩트**: 高  
**적용 난이도**: 中 (기존 메타 구조와 동일)  
**현재 키워드 노출 약점**: ASC keywords 기존 54자 → 91자로 대폭 확장, long-tail(혈압·당뇨·약달력·홍삼) 전무 → 추가

근거:
- 복약 알림 카테고리는 **검색 의도가 명확** (만성질환자, 자녀, 노년층) → 전환율 높음
- 기존 draft 가 한도 60% 만 쓰고 있어 **여백에서 빠른 임팩트** 가능
- `pill reminder` 영문 추가 → 외국인 거주자·외국어 설정 폰 사용자 흡수
- 경쟁 앱(오늘의약, Medisafe) 대비 **한국어 longtail 공백** 실재

#### 2위 — 단어요

**예상 다운로드 임팩트**: 中~高  
**적용 난이도**: 中  
**현재 키워드 노출 약점**: ASC keywords 2026-05-03 기준 70자·18개 → 87자·17개로 정밀 교체 (중복 제거 + 고효율 키워드 교체)

근거:
- 영어 학습 시장 자체가 큼, 상황별 단어(회의·여행·면접) 접근이 경쟁 앱과 **명확하게 구분**
- `보카`, `플래시카드`, `암기` 는 1위 앱(암기고래)이 점령한 키워드지만 부재 시 노출 0 → 추가
- Play short 대안("영단어 암기 단어장. 회의·여행·면접…")이 기존보다 키워드 3개 더 박혀 검색 매칭 유리
- `오프라인` 차별점 키워드는 경쟁 약함 → 틈새 공략 가능

#### 3위 — 더치페이

**예상 다운로드 임팩트**: 中  
**적용 난이도**: 中 (단, 실제 앱 기능 대조 필요 → 본진 확인 후 반영)  
**현재 키워드 노출 약점**: 시나리오 longtail(회식정산·여행정산·모임회비) 미등재

근거:
- 정산 카테고리 상위 7개 앱이 이미 메인 키워드 점령 → **longtail 다변화로 틈새 확보** 필요
- `차액정산`, `엔빵` 은 경쟁 약하고 단어요 USP와 직결
- **주의**: 더치페이 draft 는 소스 코드 미확인 상태에서 작성됨. 카카오톡 공유 기능 실제 존재 여부, 앱 이름 "더치페이" 포함 여부 강대종 확인 필수. ASC keywords 의 `더치페이` 중복 여부도 앱 이름 확인 후 조정.

#### 4위 — 메모요

**예상 다운로드 임팩트**: 中~低 (단기 기준)  
**적용 난이도**: 易 (ASC 키워드 신설 1회 작업이 핵심)  
**현재 키워드 노출 약점**: ASC 키워드 필드 자체가 비어 있음 (v2까지 미등재) → 신설만으로 즉시 효과

근거:
- 메모/노트 카테고리는 에스메모(1천만+ DL), Google Keep, Samsung Notes 와 정면 경쟁 → 단기 임팩트 제한
- 그러나 ASC keywords 필드가 비어있다는 것은 **곧 즉각적인 이득** — 신설 자체가 baseline 개선
- `광고없는 메모`, `다크 모드 메모` 는 경쟁 약한 long-tail이므로 점진적 유입 기대

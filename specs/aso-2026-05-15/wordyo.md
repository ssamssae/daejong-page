### ASC 키워드 필드 매핑 (한국)

ASC 는 콤마 구분 100자 fixed budget. 위 20개 중 본문에서 자연 노출되는 키워드(회의·여행·카페·면접·이메일·일상)는 description 만으로 색인 잡히므로 ASC 필드는 자체 검색량 높은 키워드 위주로 압축.

선정 13개 (82자):
```
영어,단어장,영단어,영어단어장,보카,플래시카드,암기,영어회화,비즈니스영어,여행영어,면접영어,무료,오프라인
```

자수 검증: `영어`(2)+`단어장`(3)+`영단어`(3)+`영어단어장`(5)+`보카`(2)+`플래시카드`(5)+`암기`(2)+`영어회화`(4)+`비즈니스영어`(6)+`여행영어`(4)+`면접영어`(4)+`무료`(2)+`오프라인`(4) = 46자 + 콤마 12 = 58자 ... 한글 1자 = ASC 카운트 1자 기준으로 58자. UTF-8 byte 카운트 적용 시(한글 3byte) 약 150byte 인데 ASC 는 character count.

상황별 카테고리 — 오늘 회의가 있으면 Meeting 카테고리만, 내일 비행기 타면 Travel 카테고리만. 필요한 영어 단어만 꺼내서 공부하세요.

한글 발음 표기 — 발음 기호(IPA) 없이 한국어로 직접 적어요. pronunciation → "프로넌시에이션" 식. 영어 초보자도 바로 소리내어 읽을 수 있어요.

완전 무료 — 광고 없음, 계정 없음, 구독 없음. 앱을 열면 바로 영단어가 나와요.

오프라인 동작 — 모든 영어 단어 데이터가 앱 안에 있어요. 인터넷 없이도 완전히 동작합니다. 비행기, 지하철, 해외 로밍 0 환경에서도 그대로.

■ 어떤 상황에 써보세요

회의 전 5분 — 오늘 영어 회의가 잡혔을 때. 비즈니스 영어 카테고리를 한 장씩 넘기다 보면 align, deliverable, stakeholder 같은 단어가 눈에 익어요.

### 차별점 surface 검증

단어요 vs 경쟁 앱(암기고래·메모리워드·말해보카) 차별점이 description 본문에서 명확히 surface 되어야 ASO 효과 극대화. 본 draft 의 차별점 4개:

- **상황별 180단어** (장면 큐레이션 → 빈도순 단어장과 다름)
- **무료·광고 0** (대부분 경쟁 앱이 광고 또는 구독)
- **오프라인 완전 동작** (비행기·로밍 환경)
- **계정 0** (가입 절차 0)

### 제외한 키워드

- `토익` / `수능` / `토플` — 단어요는 시험용 아님. 시험 키워드 박으면 사용자 기대 mismatch + 다운로드 후 이탈률 ↑.
- `AI` / `챗봇` — 런타임 LLM 호출 0 이므로 false 마케팅 금지.
- `발음` / `회화 연습` — 발음 표기는 있으나 TTS·STT 없음. mismatch 위험.
- `퀴즈` / `테스트` — 카드 학습만, 퀴즈 모드 없음.

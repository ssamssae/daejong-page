## 더치페이 USP 검증

audit 보고서(`~/daejong-page/specs/audit-2026-05-15/dutchpay.md`)와 ASO 보고서(`~/daejong-page/specs/aso-2026-05-15/dutchpay.md`)에서 추출한 더치페이의 객관적 사실:

| 특성 | 더치페이 | 경쟁사 군 비교 |
|------|---------|---------------|
| **권한 요청** | 0개 (Info.plist 위치/카메라/알림/연락처 일체 없음) | A 군: 보통 1~3개 (저장소 등). B 군: 카톡 SSO·송금 권한 다수. C 군: 연락처·이메일 SSO. **→ 더치페이 압도적 강점** |
| **회원 가입** | 0 (앱 즉시 실행) | A 군: 일부 광고 식별자만, 가입 0. B 군: 카카오 로그인 필수. C 군: 이메일 가입 필수. **→ A 군과 동급, B/C 대비 강점** |
| **즉시 계산** | 1탭 (금액→인원→계산) | A 군: 동등. B/C 군: 회원·그룹 생성 단계 필요. **→ A 군과 동급, B/C 대비 강점** |
| **다회·다인 추적** | X (단발 계산만) | A 군: 일부(고마운 정산 = 차액·차수). C 군: 핵심 기능. **→ 약점** |
| **카톡 공유** | X (현재 미구현) | B 군: native. A 군: 일부. **→ 약점, 1.1 후보** |
| **다중 통화** | X (KRW 단일) | C 군: 핵심 기능. **→ 약점 (해외 진출 시), 국내 사용엔 무관** |
| **iOS-only** | O (README 정책) | A 군: 멀티 OS. B/C 군: 멀티 OS. **→ 정책상 약점이지만 강대종 운영 토큰 ₩30K 회피와 trade** |
| **광고** | O (AdMob BANNER) | A·B·C 다양. **→ 동등, 수익 모델 정상** |

**USP 한 줄**: **"권한 0 · 가입 0 · 1탭 1/N · iOS-native · 토종 한국어"** — A 군과는 ASO 경쟁이지만 product 자체로는 토종 1탭형 계산기 카테고리의 표준에 부합. B/C 군과는 정공 충돌 X — 사용자 의도 자체가 다른 세그먼트.

## 포지셔닝 제안

**Best fit 세그먼트** (의도 적합도순):
1. **회사 회식 단발 정산** — 당일 자리, 1탭 끝. 더치페이 압도적 fit. ASO 키워드 "회식정산" longtail 강조.
2. **친구 모임 1/N** — 동일. ASO 키워드 "모임 1/N" longtail.
3. **단발 식비 정산** — 점심·저녁 자리. fit.
4. **간단 회비 단발 계산** — 동호회 회비 1회분. fit.

**Bad fit 세그먼트** (의도 부적합):
1. **여행 다회·다인 정산** — Splitwise/Tricount/고마운 정산이 더 fit. 더치페이 "여행 정산" 키워드는 의도 vs 도구 mismatch. ASO 보고에서 이미 longtail 비중을 중급으로 둔 것 적절.
2. **결제까지 통합 송금** — 카카오페이 정산하기 영역. 더치페이 침공 불요.
3. **장기 그룹 잔액 추적** — Splitwise/Tricount 영역. 1.0.x 라인업 침공 불요.

**1.1 차별점 후보** (현 1.0.x 기준 차순위):
- **카톡 공유 출력** (수동) — 계산 결과를 카톡 메시지로 1탭 공유. A 군 일부 구현. **본격 구현 권장.**
- **차액 정산 단발** — 누가 얼마 냈는지 1탭 입력 → 누가 누구에게 얼마 보낼지 출력. 고마운 정산 점령 영역. 진입 시 정공 충돌, 보류.
- **다중 통화** — C 군 영역, 해외 진출 전엔 불요.
- **그룹·내역 저장** — C 군 영역, USP 훼손 (권한 0 → SQLite 도입 시 깨짐). 보류.

**카니발리제이션 주의**:
- B 군(카카오페이 정산하기)이 카톡 내 native 라 사용자 검색이 일부 겹침. 더치페이 ASO 는 "카카오톡" 키워드 직접 박지 말 것 (B 군과 검색 충돌, 신뢰도 손실).
- C 군(Splitwise/Tricount) 영문 키워드(`splitwise`, `split bill`) 는 ASC keywords 보조 슬롯 OK — 정공 충돌 없고 영문 사용자 보조.

## 경쟁사 상세

### Daylio

**태그라인:** "Journal, Diary and Mood Tracker" — 2탭으로 끝내는 무드 트래킹

**핵심 USP:**
무드 선택(5단계) + 활동 태그 → 자동으로 패턴·상관관계 통계 생성. 글쓰기를 최소화하면서도 데이터 깊이를 확보하는 것이 핵심. 5단계 이모지 무드 + 커스텀 활동 아이콘 + Year in Pixels 히트맵이 대표 기능.

**가격:**
- 무료 (광고 포함, Android 한정)
- 프리미엄: $4.99/월 또는 $35.99/년
- 프리미엄 혜택: 고급 통계, 무제한 목표·무드·아이콘, 자동 클라우드 백업, PIN 잠금, 광고 제거

**AI 기능:**
실질적 AI 기능 없음. "AI add-on" 언급이 있으나 실제로는 룰 기반 통계 분석 수준.

**잠금/보안:**
PIN 잠금 (프리미엄 전용). 생체인증 지원.

**데이터 정책:**
모든 데이터 기기 내 로컬 저장. 계정 불필요. Google Drive / iCloud 선택적 백업 가능.

**한줄일기 대비 강점:**
- 무드 통계·상관관계 분석이 압도적으로 깊음
- 글쓰기 부담 0에 수렴하는 UX (2탭)
- 방대한 MAU와 브랜드 신뢰도 (AppStore 4.8점, 수십만 리뷰)
- 로컬 저장 + 계정 불필요 — 프라이버시 정책 동일선상

**한줄일기 대비 약점:**
- AI 공감·답글 없음 → 감정적 연결 부재
- 텍스트 서술보다 클릭 분류 → 글로 털어놓고 싶은 욕구 미충족
- Android 무료 버전에 광고 존재
- 무드 카테고리 선택 UI가 일기 쓰는 "의식(ritual)"보다 데이터 수집에 가까운 느낌

### Day One

**태그라인:** "Your Journal for Life" — 삶의 아카이브를 위한 프리미엄 저널

**핵심 USP:**
군사급 E2E 암호화(AES-GCM-256) + 멀티미디어(텍스트·사진·영상·오디오·드로잉) + AI Daily Chat(대화로 일기 작성) + 가족·그룹 공유 저널. 진지한 저널러를 위한 올인원 플랫폼.

**가격:**
- 무료: 무제한 텍스트 항목, 일일 프롬프트, 검색, E2E 암호화, 1기기
- Silver: $49.99/년 — 다기기 동기화, 멀티미디어, 통합
- Gold: $74.99/년 — AI Daily Chat, Go Deeper 프롬프트, 항목 요약, 이미지 생성, AI 하이라이트

**AI 기능 (Gold 전용):**
- Daily Chat: AI와 대화로 하루 리뷰 후 저널 항목 자동 생성
- Go Deeper: 현재 작성 내용 기반 심층 프롬프트 자동 생성
- Entry Highlights: 항목별 주요 테마·감정 요약
- AI 이미지 생성 (항목 삽화)

**잠금/보안:**
Passcode + 생체인증 + E2E 암호화(기본값). 암호화 키는 기기에만 보관, Day One 서버 접근 불가.

**데이터 정책:**
클라우드 저장 (Day One 서버, E2E 암호화). 계정 필수. 계정 없이 로컬 전용 사용도 가능하나 동기화 불가.

**한줄일기 대비 강점:**
- 기능 깊이·미디어 지원이 업계 최고
- E2E 암호화로 프라이버시 신뢰도 최고
- AI 기능이 2026년 기준 가장 성숙함 (단순 답글이 아닌 대화형 저널링)
- Mac·iPad·iOS·Android 완전 지원

**한줄일기 대비 약점:**
- 가격 장벽 큼 (Gold $74.99/년 vs 한줄일기 ₩1,900 일회성)
- 계정 필수 → 가입 마찰
- 기능이 복잡 → "오늘 30초 안에 쓰고 싶다"는 니즈와 미스매치
- AI 기능이 Gold 구독에만 있어 진입 문턱 높음

### Reflectly

**태그라인:** "World's First Intelligent Journal" — CBT·긍정심리학 AI 저널

**핵심 USP:**
Cognitive Behavioral Therapy(인지행동치료)·마음챙김·긍정심리학을 앱 UX에 내장. AI가 이전 항목을 학습해 개인화된 질문을 생성하고 감정 상관관계를 분석. 가이드형 저널링에 특화.

**가격:**
- 무료: 제한된 기능 (7일 트라이얼 후 유료 전환 유도)
- 프리미엄: $9.99/월 또는 $59.99/년

**AI 기능:**
- 사용 이력 기반 맞춤 질문 생성
- 무드-활동 상관관계 AI 인사이트
- 긍정 심리학 기반 일일 챌린지 생성
- 감정 그래프·통계

**잠금/보안:**
생체인증 지원. 지문·얼굴 데이터 앱에 저장 안 함.

**데이터 정책:**
클라우드 저장 필수. 계정(이메일) 가입 필수. EU GDPR 준수. 정신건강 데이터는 특별 카테고리로 명시 동의 처리.

**한줄일기 대비 강점:**
- 심리학 방법론 내장 → "왜 이렇게 느끼는가" 탐구 지원
- AI가 사용자 이력을 학습해 점점 개인화
- 감정 트래킹 통계가 상세

**한줄일기 대비 약점:**
- 구독료 고가 ($59.99/년)
- 계정 필수, 데이터 클라우드 → 프라이버시 민감 유저 이탈
- 질문 나열형 UX → "한 줄 쓰고 끝내고 싶다"는 니즈와 반대
- 한국어 최적화 미흡, UI가 다소 서양 감성

### Stoic

**태그라인:** "Beat Stress in 120 Seconds — AI-Powered Journal" — 스토아 철학 기반 멘탈케어

**핵심 USP:**
일기·무드트래킹·명상·호흡운동·수면 저널·집중력 타이머를 하나의 앱에 통합. 스토아 철학(마르쿠스 아우렐리우스·세네카) 콘텐츠를 UX에 내재화. AI가 일기 패턴 분석 후 팔로업 프롬프트 생성.

**가격:**
- 무료: 기본 일기·프롬프트·확언
- Pro: $3.99/월
- Pro+AI: $9.99/월

**AI 기능 (Pro+AI):**
- 일기 항목 패턴 분석·인사이트
- AI 팔로업 프롬프트 자동 생성
- 하루 패턴 기반 개인화 알림

**잠금/보안:**
Passcode/TouchID/FaceID. iCloud 저장.

**데이터 정책:**
로컬 + iCloud 선택 동기화. iOS ↔ Android 동기화 미지원. Stoic 자체 서버 없음.

**한줄일기 대비 강점:**
- 멘탈헬스 생태계 통합 (명상·호흡·수면 등) → 올인원 웰빙
- 스토아 철학 콘텐츠로 차별화된 정체성
- Pro 가격이 합리적 ($3.99)

**한줄일기 대비 약점:**
- 철학 콘텐츠에 특화 → 일상적 감정 기록보다 "수련" 느낌
- Android ↔ iOS 동기화 미지원
- 한국 유저에게 스토아 철학은 낯선 진입점

## 한줄일기 USP 검증

경쟁사 조사를 바탕으로 한줄일기 핵심 차별점 4가지의 실제 유니크함을 평가한다.

### 1. 하루 한 줄 울트라 숏 포맷

**평가: 부분 유니크 (DayGram이 직접 경쟁)**

DayGram이 동일 포맷을 오랫동안 운영 중이다. 그러나 DayGram은 AI 기능이 없어, "한 줄 + AI 답글" 조합으로 보면 한줄일기가 유일하다. Daylio는 2탭 로그로 더 짧지만 텍스트 서술 포맷이 아니다. **"텍스트 한 줄 서술 + AI 응답" 조합은 시장에서 아직 희소.**

### 2. AI 공감·격려 답글

**평가: 국내 시장에서 유니크, 글로벌로는 경쟁 증가 중**

글로벌: Day One Gold (AI Chat), Stoic Pro+AI, Reflectly 모두 AI 기능이 있으나 구독 고가이거나 형식이 다름. 한국 국내: 마인디 등이 있으나 모두 계정 필수·클라우드·복잡한 입력 구조. **"한 줄 입력 → 즉시 AI 공감 한 문장, 계정 없음"은 국내에서 현재 유일.**

### 3. 히트맵 스트릭 시각화

**평가: 차별점 보통 (Daylio Year in Pixels 등 선례 있음)**

Daylio의 "Year in Pixels", DayGram의 연간 뷰 등 이미 다수 경쟁자가 히트맵 유사 시각화를 제공한다. **차별점으로 내세우기보다 "있어야 하는 기본값"으로 포지셔닝이 적절.**

### 4. 계정 없음 / 로컬 우선

**평가: 가치 있는 차별점, 그러나 경쟁사 일부도 동일**

Daylio, DayGram, Stoic도 계정 없이 사용 가능하다. 그러나 AI 기능이 있는 경쟁자 중에서 계정 없이 AI를 쓸 수 있는 앱은 한줄일기가 거의 유일하다. **"AI 기능을 쓰기 위해 계정을 만들지 않아도 된다"는 메시지가 핵심.**

## 다음 액션

### 1. "AI + 로컬 + 계정없음" 삼각형을 스토어 메시지 전면에 배치

현재 스토어 부제 방향이 맞다. 스크린샷에서도 "계정 없음 + AI 답글 + 로컬 저장" 세 가지를 세 장의 스크린샷에 각각 하나씩 시각적으로 명시하는 것을 권장한다. "AI 일기 계정없음" 키워드가 ASO 랜딩될 수 있도록 최적화.

### 2. DayGram과의 차별화 — AI 답글 경험을 온보딩 첫 화면으로

DayGram이 동일 포맷을 선점하고 있으므로, 한줄일기는 첫 실행 시 AI 답글 경험을 즉시 보여줘야 한다. 샘플 일기 → 샘플 AI 답글 → "지금 써보세요" 흐름으로 온보딩 개선. DayGram 대비 차별점이 즉시 체감되도록.

### 3. 한국어 감정 표현 특화 AI 프롬프트 고도화

마인디 등 국내 경쟁자들은 한국 정서에 맞는 AI 응답을 강점으로 내세운다. 현재 GPT-4o-mini 시스템 프롬프트에 한국적 공감 표현 패턴을 명시적으로 지시해 감성 품질을 높인다. Worker의 system prompt를 A/B 테스트할 수 있는 구조 마련.

### 4. PDF 내보내기 또는 연간 회고 기능 추가 검토 (1.2.0 후보)

DayGram의 "PDF 북 내보내기"는 유저 리텐션에 기여하는 강력한 기능이다. 한줄일기도 연말 회고 시즌(12월)에 "올 한 해 내 한 줄들" 형태의 간단한 공유 카드 생성 기능을 추가하면 바이럴 루프와 리텐션을 동시에 잡을 수 있다.

### 5. AI fallback 구현 (BACKLOG P1 병행)

Cloudflare Worker 단에서 OpenAI timeout 시 캐시된 위로 문장(오프라인 fallback pool) 반환 → "AI가 잠시 바빠요, 그래도 오늘 기록은 소중해요" 메시지를 표시. 경쟁사 대비 서비스 안정성 신뢰도 향상. 현재 `_offlineStub()` 구현의 Worker 단 버전.

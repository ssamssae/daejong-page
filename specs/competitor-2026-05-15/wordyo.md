## 단어요 USP 검증

### 셀링포인트 4개

1. **장면별 큐레이션 (회의·여행·카페·일상·이메일·면접 × 30단어)**
2. **무료, 광고 0, 계정 0**
3. **오프라인 완전 동작 (외부 서버 호출 0)**
4. **카드 플래시 + 한글 발음 + 한 줄 응원**

### 시장 검증 — 각 셀링포인트별 경쟁 강도

| USP | 시장에 있나? | 단어요 차별 강도 |
|-----|-------------|----------------|
| 장면별 큐레이션 | **거의 없음** — 메모리워드 커뮤니티에 "비즈니스 영어 표현 모음" 정리글은 있으나 앱 자체가 6 카테고리 × 30단어 큐레이션 제공은 0. 토익단어/비즈영어 앱이 시험·주제별 분류는 있으나 "회의 5분 전 / 비행기 게이트 앞 / 면접 당일 아침" 같은 **순간 단위 큐레이션** 0. | **고** — 본 USP 가 단어요의 가장 큰 차별점. 시장 공백 있음. |
| 무료 광고 0 계정 0 | 부분적 (똑똑단어장만 광고 0 / 메모리워드·암기고래는 광고 ON / Anki iOS 유료) | 중 — 똑똑단어장과 직접 경쟁. 다만 똑똑단어장은 큐레이션 0 이라 USP 1 과 결합 시 단어요 우위. |
| 오프라인 동작 | 대부분 앱이 데이터 저장은 로컬이나 광고 SDK·분석 SDK 로 네트워크 호출 함. 진짜 "외부 서버 호출 0" 은 드뭄 (Anki desktop, 단어요) | 중 — 비행기·로밍·지하철 사용자에게 명확한 셀링. 단, 사용자 인식 부족 (검색 키워드 "오프라인 단어장" 검색량 중) |
| 카드 플래시 + 한글 발음 | 카드 플래시는 표준 (Quizlet, Anki, 토익단어 다 있음). 한글 발음 표기는 일부 앱만 (IPA 위주가 다수) | 저~중 — 카드 자체는 평범. 한글 발음은 초중급 사용자에게 어필 |

### Best-fit 사용자 세그먼트 (작은→큰)

1. **🎯 직장인 / 영어 회의·이메일 빈도 사용자** — "회의 5분 전" / "이메일 회신 직전" 사용 후크. 비즈니스영어 카테고리 + 이메일 카테고리. **단어요 가장 강력한 fit**.
2. **🎯 출장·여행 빈도 사용자** — "여행 영어" 카테고리 + 오프라인 동작 (비행기·로밍 0 환경). 게이트 앞 30분 사용 후크.
3. **🎯 면접 준비 단기 사용자** — "면접 당일 아침 10분" 후크. 면접 카테고리 30단어로 단기 집중. 사용자 라이프타임 짧지만 진입 강함.
4. **🎯 광고·계정·구독 피로 사용자** — "단어장 앱 여러 개 써봤는데 결국 안 쓰게 됨" → 단어요의 0-friction 진입.
5. **🎯 초중급 영어 사용자** — 한글 발음 표기로 IPA 부담 0. 초고급 사용자 (Anki 류) 와 비경쟁.

### Worst-fit 사용자 (단어요 안 맞음)

- 토익·수능·텝스 등 **시험 점수 목표** 사용자 — 단어요는 시험 빈도 큐레이션 0. ASO 키워드에서 토익/수능 의도적 제외 (audit-2026-05-15 메모 검증).
- **8만+ 단어 마스터** 목표 사용자 — 180 단어로 절대 부족. 암기고래/Anki 영역.
- **다국어 학습자** — 영어만 지원.
- **음성 청취 학습자** — TTS 미탑재.

## 포지셔닝 제안

### Core positioning statement

> "시험용이 아니라, 그 순간 쓸 단어장."
> 영어 회의 5분 전, 비행기 게이트 앞, 면접 당일 아침 — 그 순간에 같이 떠오를 6개 상황 × 30단어.

이 포지셔닝은 **8개 경쟁 앱 중 0개가 점유**. 단어요가 카테고리 정의자(category definer)로 갈 여지 있음. 시장이 작더라도 (검색량 "상황별 영어 단어" 중) 한 번 자리잡으면 카테고리 1위.

### 포지셔닝 좌표 — 2개 축

```
                  (시험 점수 목표)
                         |
                         |
        Anki ●           |        ● 토익단어
                         |        ● 메모리워드
                         |        ● 암기고래
                         |
(체계적 학습) ---------------+--------------- (즉시 사용)
                         |
                         |
        Quizlet ●        |
        말해보카 ●        |        ● 단어요 ★
                         |        (장면별 + 그 순간)
        Drops ●          |
        똑똑단어장 ●      |
                         |
                  (일상 회화 목표)
```

단어요 = "즉시 사용" + "일상 회화 목표" 사분면. 같은 사분면에 Drops 가 있으나 일러스트·게임화 vs 단어요는 텍스트·미니멀 — 서로 다른 사용자 취향.

### 메시지 우선순위 권장 (ASO description 기준)

1. **장면별 큐레이션 우선** (현재 ASO description 이 이미 채택 — 유지)
2. **무료·광고 0 그 다음** — 메모리워드/암기고래 광고에 지친 사용자 후크
3. **오프라인 동작은 세 번째** — 비행기·로밍 환경 사용자에게만 즉시 어필, 평균 사용자에겐 약함
4. **한글 발음은 부차** — 초중급 사용자에게만 어필. headline 배치 X

### 다음 액션

1. **단기 (1~2주)** — Play Console **Custom store listing** 으로 A/B 테스트. A안 = 현재 description (장면별 우선) / B안 = 보상·게임화 강조 (메모리워드 류 후크). 4주 후 conversion rate 비교. *비가역 액션 — 강대종 ack 필요*.
2. **단기 (1~2주)** — Apple Search Ads 한국 캠페인 (USP 1 키워드 "상황별 영어 단어" / "비즈니스 영어 단어장") 5만원 budget 으로 CPI 테스트. *비가역 — ack 필요*.
3. **중기 (1~3개월)** — 카테고리 7번째 추가 검토 — "운동·헬스장 영어", "병원·약국 영어", "데이팅 영어" 중 1개. 사용자 요청 시그널 수집 후 결정.
4. **중기 (1~3개월)** — 영어권 사용자용 역방향 버전 (한국어 → 영어 단어장, 출장자 대상) 별 앱? 또는 카테고리 추가? 별 패키지로 가는 게 깔끔.
5. **장기** — 메모리워드 류 보상 모델 도입 검토 — 그러나 단어요 USP "광고 0, 보상 후크 0, 깔끔" 과 정면 충돌. 하지 않는 게 정합. *결정 보류*.

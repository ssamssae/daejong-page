## 약먹자 USP 검증

### USP-1: "휠 UI 5초 등록"
- **시장 검증**: ✅ 차별성 있음. 경쟁 앱 대부분이 약 이름 텍스트 + 시간 picker + 빈도 라디오 + 메모 텍스트 등 4~6단계. 약먹자는 약이름 + 휠 용량 + 휠 시간 3단계, 0.5알 단위 휠로 정밀.
- **위험**: "5초"는 단발 약 등록 기준. 다회 시간(아침/저녁 등) 추가 시 시간 picker 반복 → 실제 5초 X. 마케팅 문구 보완 필요 ("첫 약 5초, 시간 추가 1초씩").

### USP-2: "100% 로컬 전용 + 광고 없음"
- **시장 검증**: ⚠️ 코드 실태와 부분 모순 (audit 결과 참조). 광고: iOS AdMob 통합 + 배너 노출. 로컬: Hive 평문 + Android 자동 백업 미차단.
- **본 사이클 패치 (S1+S2)**: Android allowBackup=false + dataExtractionRules 차단 + 알림 평문 일반화 → 정합 1차. AdMob 잔류는 S3 별도 사이클.
- **확정 후 시장 메시지**: "광고 없음 (iOS 한정 배너 1개만, 광고 식별자 추적 X)" + "데이터 100% 로컬" 로 솔직하게. Medisafe 유료·MyTherapy 광고 데이터 공유 대비 강한 차별.

### USP-3: "다크 테마 + 군더더기 0"
- **시장 검증**: ✅ 차별성 있음. 오늘의약·아이약·커넥트케어 모두 라이트 테마 위주. 약먹자 단일 다크 = 야간 복용 시 (수면 직전 약) 눈부심 0.
- **위험**: 노년층은 라이트 선호 가능성. 1.0.4+ 라이트 옵션 검토 후보.

### USP-4: "광고/계정/가입 0 → 노년층 즉시 사용"
- **시장 검증**: ✅ 강력. 오늘의약·아이약·커넥트케어 모두 계정 가입 또는 카메라/사진 권한 요구. 약먹자는 알림 권한만 → 노년층 자녀 케어 시나리오에서 진입장벽 압도적 낮음.
- **포지셔닝 후보**: "자녀가 부모 폰에 5초 만에 설치하는 약 알람" — Play 짧은 설명·ASO 부제에 박을 가치.

## 포지셔닝 제안

### 1차 타겟 세그먼트 (best-fit, ARPU 0 무관 — 입소문·평점 우선)
- **혼자 영양제 챙기는 30~50대 직장인** — 종합비타민·오메가3·홍삼 매일. 광고 거부감 강함. iOS 비중 높음 (Medisafe 유료 전환 흡수).
- **부모 약 챙겨주는 자녀** — 광고/가입 없는 가벼운 앱 폰에 설치 → 부모는 알림만 받음. 노년층 직접 검색 비율 낮으므로 자녀 채널이 주력.

### 2차 타겟 세그먼트 (확장)
- **만성질환자 (혈압·당뇨)** — 매일 정시 복용. 다만 본 그룹은 의료 신뢰성 (아이약·커넥트케어) 더 선호 → 약먹자가 정면 경쟁 어려움. 보조 앱 포지션.

### 비추 타겟
- **가족 caregiver, 약사·의료 종사자** — Medisafe·아이약·커넥트케어 영역. 약먹자 기능 부족.
- **임신·피임 추적** — 약 알림 태양이가 인지도 선점. 약먹자 굳이 진입 X.

### 마케팅 메시지 후보 (Play 짧은 설명 80자 / ASC Subtitle 30자 활용)
- **A안 (가벼움 강조)**: "약·영양제 복용 시간 알림 5초 완성. 광고·계정 없음, 100% 로컬. 부모님 폰에 부담 없이."
- **B안 (영양제 진입)**: "종합비타민·오메가3·홍삼 매일 챙기는 가장 가벼운 약 알림. 5초 등록, 광고 없음, 다크 테마."
- **C안 (만성질환)**: "혈압·당뇨·영양제 — 매일 정시 복용을 가장 단순하게. 광고 없음, 가입 없음, 로컬 전용."

권장: **A 또는 B**. C 는 의료 신뢰성 경쟁사가 강해 정면 어려움.

## 한 줄 요약
Anthropic이 Opus 위 티어로 개발 중인 미공개 모델 "Mythos"의 API 식별자가 잠깐 노출돼 공개 임박설이 돌았으나, 발화자는 Anthropic의 "일반 공개 계획 없음" 발언·IPO·OpenAI 경쟁 정황을 근거로 당분간 공개 안 되고 능력만 차기 Opus에 흡수될 것이라 본다.

## 픽업
- Mythos는 현재 최상위 Opus 4.8 위에 얹히는 별도 신규 모델 패밀리로, 유출 초안에서 "step change·역대 최강"으로 불렸다. 특히 사이버보안(취약점 탐지·패치)에 비정상적으로 강한데, 같은 능력이 침투·해킹에도 쓰여 위험하다.
- 배포는 invite-only "Project Glasswing"으로 검증된 사이버보안 기업·정부에만 풀렸고, 4월 50개 파트너에서 최근 150개 조직·15개국 이상으로 확대됐다. 다만 공개된 벤치마크 1건에선 이미 쓸 수 있는 GPT 모델과 막상막하라 "역대 최강"은 IPO·마케팅 hype일 수 있다.
- 오늘 Mythos 식별자가 API에 몇 분간 떴다 사라지며 스샷·hype가 폭발했는데, 발화자는 이런 유출이 의도된 마케팅(흘렸다 회수하면 X·링크드인이 공짜로 떠들어줌)일 가능성에 회의적이다.
- Anthropic은 4월에 "Mythos preview를 일반에 공개할 계획이 없다"고 자기 입으로 명시했고 이후 이를 번복한 적이 없다. 즉 지금 난리난 그 게이트 버전이 공개되는 게 아니라는 뜻이다.
- 타이밍이 묘하다 — 6/1 Anthropic이 약 9,650억 달러(거의 1조) 밸류로 비공개 IPO를 신청했고, 3일 뒤 "AI가 스스로 개선되기 시작했으니 속도를 늦춰야 할 수도"라는 보고서를 냈다. 주식 팔기 직전 "우리가 가장 위험한 모델을 갖고 있다"고 상기시키는 건 좋은 스토리이며, hype에는 동기가 있다.
- 가장 큰 변수는 OpenAI 경쟁이다. GPT 5.6 임박설(베팅시장 6월 내 출시 80%+)이 있고, 두 회사는 릴리스를 서로 위에 쌓아왔다(OpenAI 코딩모델 출시 몇 분 만에 Anthropic 대응 / OpenAI Codex 2개월 무료 프로모션 당일 Anthropic이 Claude 한도 50% 상향). 예상 시나리오는 GPT 5.6이 나오면 Anthropic이 며칠 내 맞불을 놓는 것이다.
- 발화자의 베팅: 로그인해 쓰는 공개 "Mythos"는 당분간(어쩌면 그 이름으론 영영) 안 나오고, 핵심 능력은 안전장치가 충분해지면 차기 Opus에 조용히 흡수될 것이다. 따라서 기다릴 건 거창한 Mythos 출시가 아니라 다음 Opus + GPT 5.6이 Anthropic 손을 강제하는지다. (Glasswing 프리뷰 가격은 100만 토큰당 입력 $25·출력 $125로 현 Opus의 5배.)

## 용어
- **Mythos** [모델 · 구독]: Anthropic이 Opus 위 티어로 개발 중이라 알려진 미공개 차세대 모델 패밀리 코드명.
- **Project Glasswing** [워크플로우 · 문화]: Anthropic이 Mythos를 검증된 사이버보안 파트너·정부에 invite-only로 배포하는 프로그램.
- **frontier AI** [모델 · 구독]: 최첨단·최대규모의 선도 AI 모델을 가리키는 표현(프런티어 모델).
- **step change** [모델 · 구독]: 점진적 개선이 아닌 비약적 성능 도약을 뜻하는 말로, Anthropic이 Mythos를 자칭할 때 쓴 표현.
- **input/output token pricing** [빌링 · 운영]: API 비용을 입력·출력 토큰 100만 개당 달러로 매기는 방식 (Mythos 프리뷰 $25/$125 per M).
- **prediction(betting) market** [기타]: 사건의 발생 시점·여부에 돈을 거는 예측시장으로, 모델 출시 시기 확률 추정 근거로 인용됨.
- **GPT 5.6** [모델 · 구독]: OpenAI 차기 모델로 예상되는 버전(임박설).
- **Codex** [도구 통신 (MCP · CLI · API)]: OpenAI의 코딩 에이전트/CLI 제품 (2개월 무료 프로모션으로 언급).

## 3way 왕복 테스트

**맥미니 → 맥북 방향:**
- mac-report.sh로 맥미니가 맥북 tmux에 명령 paste → 맥북 Claude 처리 후 SSH로 맥미니에 [결과] 전송 성공
- 단, mac-report.sh 래퍼가 자동으로 `agent-msg-notify macmini macbook 결과 $TITLE` 호출하는 버그 발견 → [결과] 노이즈 메시지 발생

**맥북 → 맥미니 방향:**
- `agent-msg-notify.sh macbook macmini 명령 "..."` → 맥미니 채팅방 도착 ✓
- SSH로 맥미니에서 `agent-msg-notify.sh macmini macbook 결과 "..."` 실행 → 맥북 채팅방 도착 ✓

**WSL → 맥미니 방향:**
- WSL이 [명령] 전송은 성공, 맥미니 자율 응답 불가
- 원인: 5/3 launchd 전부 제거 후 inbox listener 미설치
- 결정: 옵션 b (본진 SSH 코디 패턴) 유지. 새 LaunchAgent 불필요

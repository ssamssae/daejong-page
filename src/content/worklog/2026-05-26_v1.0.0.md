- **부정확 가정은 노드가 catch**: 본진 디렉티브에 "settings.json git tracked 가역" 박았는데 노트북이 `~/.claude` 가 git repo 아니라고 catch. 다른 노드에 동일 가정 적용했으면 silent fail 가능. 디렉티브 본문 일반화 가정 줄이고 백업 + Edit diff 의 가역 경로로 통일.

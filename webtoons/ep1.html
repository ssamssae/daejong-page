새벽 5시, 강대종

5노드 mesh 자기소개 4컷

자기소개

안녕하세요, 1인 앱 개발자 강대종입니다. 메모요·단어요·한줄일기 같은 작은 앱을 만들고 있어요. 이 첫 웹툰은 제가 평소에 어떻게 일하는지 4컷으로 풀어본 거예요.

1컷 — 새벽 5시

저는 잠귀가 밝은 편이라 새벽에 자주 깨요. 그럴 때 침대에 누운 채로 텔레그램 봇한테 메시지 한 줄 보냅니다. "5노드 빌드 가자."

이 한 줄이 트리거예요. 폰에서 보낸 메시지가 맥북 본진 챗봇한테 들어가요.

2컷 — 자는 동안 팀이 켜진다

저는 컴퓨터를 다섯 대 굴립니다.

  • 🍎 Mac 본진 — 지휘관, 모든 결정·검증·머지를 담당하는 팀장
  • 🪟 WSL Ryzen — 낮 즉응 작업자
  • 🏭 Mac mini — 빌드·배포 엔진 + Claude Code 노드
  • 🖥 데스크탑 (RTX 3060Ti) — GPU 노드, 이미지/SD 실험
  • 💻 노트북 (RTX 3060) — 외출용 + 5번째 Claude Code 노드

다섯 대 다 자기 AI 에이전트(Claude Code)가 돌고, 텔레그램 봇으로 묶여있어요. 본진이 "이거 해" 라고 명령하면 노드들이 동시에 깨어나서 일을 시작합니다.

3컷 — 코드 → 빌드 → 배포

명령 한 줄이 들어가면 그 다음은 컨베이어벨트입니다. 코드 작성 → 테스트 → 빌드 → 서명 → 스토어 업로드 → 발행. 사람은 졸려도 파이프라인은 또렷해요.

각 노드가 자기 prefix 브랜치(mac/, wsl/, macmini/, desktop/, notebook/)에서 일하고, 본진이 PR 머지로 통합합니다. 충돌은 거의 없어요. 같은 파일·같은 브랜치를 동시에 안 만지도록 룰이 박혀있거든요.

4컷 — 그리고 다시 잠

배포 완료 알림이 폰으로 와요. 저는 다시 잠듭니다.

이 첫 웹툰은 제가 이런 식으로 산다는 자기소개이자, 앞으로 이 컬렉션에서 1인 개발 일상을 4컷씩 풀어볼 거라는 예고편이에요.

## 밤 세션 — 5노드 입력중 훅 전수 점검

- 형님 "5노드 다 입력중 안 뜬다" 지적 → 본진부터 진단. 본진 데몬(PID 63408) http=200 연속 + 수동 sendChatAction ok:true 확인, 메커니즘 정상.
- 나머지 4노드(🪟 WSL / 🏭 맥미니 / 🖥 데스크탑 / 💻 노트북)에 자가점검 디렉티브 발사(API 비용 사전 경고 + 형님 ack 후). 4노드 전부 (a) hook 배선 O (b) 데몬 http=200 O (c) sendChatAction ok:true O 보고.
- 결론: 5노드 전부 정상, 통일 패치 불필요. 비-버그 동작 2건 확인 — ① 입력중은 prompt source 가 telegram 인 turn 에만 뜸(노드 간 tmux/SSH 디렉티브 turn 엔 안 뜸, source 게이트), ② 데몬은 per-turn 방식이라 turn 사이 idle 엔 표시 없음. 형님 초기 "안 뜬다"는 데몬 활성 직전 짧은 타이밍 perception 으로 결론, "이제 나온다" 확인.

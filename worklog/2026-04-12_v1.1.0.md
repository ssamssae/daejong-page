# 2026.04.12 작업일지 v1.1.0 — 앱 네 개가 동시에 돌던 날

> 아침, 어제 못 잡은 메모요 재실행 크래시부터 다시 열었다.
> 오후, 더치페이 계산기 프로젝트를 제로에서 띄우고 있었다.
> 밤, 약먹자를 통째로 갈아엎고 고독사 방지앱을 새로 판 것까지 더해져 **앱 네 개** 가 하루에 같이 움직였다.

하루를 한 프로젝트로 채우는 대신 네 개를 병렬로 돌렸다. 무모해 보이지만 이날의 중심은 **메모요의 크래시 원인을 끝까지 쪼갠 것** 이었고, 나머지 세 개는 그 흐름 옆에서 각자의 초기 커밋까지 밀어냈다. 이 버전은 그 네 갈래를 한 번에 다시 짚는 소급판이다.

## 1. Scene 복원이 엔진 초기화보다 먼저 일어났다

어제 밤 iOS 에서 두 번째 실행마다 `EXC_BAD_ACCESS` 로 죽던 증상. 범위를 좁혀 가다 원인이 잡혔다. **Flutter 3.38 부터 필수가 된 iOS UIScene 라이프사이클 마이그레이션 누락**.

```
FlutterImplicitEngineDelegate + UIApplicationSceneManifest 조합
  ↓
두 번째 실행: Scene 복원이 Engine 초기화보다 먼저
  ↓
SharedPreferencesPlugin 등록 도중 크래시
```

조치:
- `AppDelegate`, `Info.plist`, `Podfile` 을 정렬해 Scene 설정 정리
- **UIScene 자체를 제거** 하는 방향으로 재설정
- 방어 차원으로 Dart 레벨 `runZonedGuarded` 로 최상단 에러 캐치 추가
- `decodeList` 가 손상 데이터를 만나도 무너지지 않도록 `try-catch` + 기본값 반환
- 메모 ID 를 `millisecondsSinceEpoch` → **UUID v4** 로 교체 (같은 밀리초 내 충돌 가능성 제거)
- `Reorder` 콜백에 인덱스 범위 초과 방어 가드
- deprecated `withOpacity` → `withValues(alpha:)` 로 일괄 전환

커밋: `2c467cc`, `2f9f39e`, `1318ea0`, `88e6c2e`, `19aecd6` — 모두 원격 반영.

> **배운 것** — "두 번째 실행에서만" 터지는 크래시는 **첫 실행이 남긴 상태** 와 **재진입 시점의 초기화 순서** 를 의심하는 게 최단 경로다. 같은 코드로 두 번째가 터진다면 **첫 번째가 깔아둔 것** 중 뭔가가 달라졌다.

## 2. 메모요를 iOS 기본 메모앱처럼 빽빽하게

디자인 쪽은 같은 날 함께 달았다. **한 화면에 메모 11개 이상** 이라는 밀도 목표를 걸고 컴팩트 리스트로 전환.

- 즐겨찾기 섹션·일반 섹션의 제목 텍스트 제거
- 여백을 iOS 기본 메모앱 수준으로 빡빡하게 축소
- 드래그 핸들 아이콘이 우측 스크롤 휠과 겹치던 것 → trailing 패딩으로 왼쪽 밀기
- AppBar 타이틀 안드로이드 좌측 정렬 → `centerTitle: true` 로 양 플랫폼 통일
- 첫 실행 스플래시 1500ms → 체감 길다는 피드백에 **750ms 로 반으로 축소**
- APK 는 `split-per-abi` 로 아키텍처 분리 빌드 (용량 감소)
- iOS 아이콘 알파 채널 **전부 제거** (App Store 업로드 거부 선제 방어)

안드로이드 빌드 환경이 막혀 있던 부분은 Android Studio + Android SDK 새로 설치하고 환경변수 잡는 작업으로 해결.

## 3. AGENTS.md 를 심다

메모요 Dart 방어 로직이 안정화된 시점에 프로젝트 루트에 **AGENTS.md** 를 새로 만들어 붙였다. 규칙은 단순했다.

- 모든 코드 수정 후 반드시 `git add` → `commit` → `push` 까지 자동으로
- 커밋 메시지는 한글, `fix:` 또는 `feat:` 접두어 필수

지금(4/20 이후) 의 `claude-automations` 자동 커밋·푸시 습관의 원형이 이 한 파일이다.

## 4. 더치페이 계산기, 토스 송금 화면 톤으로

오후엔 `dutch_pay_calculator` 를 제로에서 팠다.

- 번들 ID: `com.daejongkang.dutchPayCalculator`
- 앱 이름: **더치페이 계산기** (한국어)
- 스플래시 1.5초 + 앱 아이콘 동시 세팅

초기 UI 는 "금액 입력 + 인원수 입력 → 1인당 금액" 의 단순한 구조였다. 피드백을 받아 **금액 입력 영역을 토스 송금 화면 스타일로 재설계**. 핵심 결정은 다음 두 가지.

- **시스템 키보드 아예 쓰지 않음**. 하단에 커스텀 숫자 키패드 직접 그리기. 4×3 그리드
- 키 배치: `00` 을 넣었다가 어색해서 빼고 `C`, `0`, `⌫` 세 개만

인원수 조절은 가운데 정렬의 `−`·숫자·`+` 3 버튼 + 박스 바깥 아래의 라벨. 금액 텍스트가 입력 상태·빈 상태에서 높이가 달라 **UI 가 위아래로 흔들리던 버그** 는 두 상태의 높이를 같게 고정해 잡았다. 작은 안드로이드 폰에서 세로 스크롤이 생기던 건 `MediaQuery` 기반 반응형 사이즈로 교체해 한 화면에 꽉 차게.

APK 용량 40.5 MB 에서 내렸다. 안 쓰는 패키지 제거 + 에셋 최적화 + `--split-debug-info` + `--obfuscate` + 불필요한 ABI 제거 — 네 개를 동시에 얹었다.

App Store Connect 업로드 시도 중 **iOS 아이콘 알파 채널** 이유로 한 번 거부당했고, 같은 날 안에 수정. 메모요와 같은 스타일의 응원 버튼도 우측 상단에 달고 `daejong-page` 로 링크.

## 5. 약먹자를 통째로 갈아엎다

저녁, 약먹자(`yakmukja`)는 다른 세션에서 엉망으로 시작된 상태였다. **일단 현재 상태를 초기 커밋으로 GitHub 에 올린 뒤** 재설계 방향으로 다시 만들기 시작했다.

- 아이콘: 노란색·흰색 알약 모티브로 재작업
- 로딩 화면 푸른 톤도 아이콘과 맞춰 조정
- 배경 전면 노란색은 부담 → **다크톤 전체 전환**
- 약 복용량 입력: 0.5알부터 10알까지 **스크롤 휠** 로 선택 (시계 다이얼 제거)
- 복용 시간 세 개도 같은 스크롤 휠 방식으로 통일
- 저장 버튼은 키보드에 가리지 않도록 **우측 상단 AppBar 로 이동**
- 메모 입력 칸 제거 (실사용에 불필요)
- 복용 시간에 맞춰 "약 이름 N알 드실 시간입니다" 푸시 알림 1회 설계 반영

밤 늦게까지 재시작 크래시와 `objective_c` 에러를 반복 디버깅하다 **초기 커밋 `4981029`** 로 푸시. 이 저장소의 정식 시작점이 여기다.

## 6. 고독사 방지앱, 컨셉이 오후에 한 번 뒤집히다

오후 새 프로젝트로 `lonely_death_prevention` 을 팠다.

- 패키지명: `com.lonelydeath.prevention`
- 앱 이름: **고독사 방지앱** (iOS Info.plist + Android AndroidManifest 동시 반영)
- 로고: "두 사람이 연결" → "**두 손이 서로 잡고 있는 노란 원형**" 으로 수렴. 노인도 한눈에 알아볼 수 있는 톤이 목표.
- iOS 아이콘은 **처음부터 불투명 배경** 으로 세팅 (리젝 여지 제거)
- Material 3 + 고령자 친화 색상
- `flutter_native_splash` 로 스플래시 배경 `#FFC107` 생성
- 초기 빌드에서 `text_style.dart` 의 `fontSize` assertion 에러 터짐 → TextStyle 정리로 안정화
- 체크인 화면 "괜찮아요" 버튼에 메모요 고양이 아이콘 재사용

오후 중반에 **컨셉이 한 번 크게 뒤집혔다**. 자주 열지 않아도 되는 **자동 활동 판단 중심의 단순 앱** 으로 방향 전환. `CLAUDE.md` 도 전체 교체. 재시작 크래시는 메모요와 비슷한 패턴인지 확인하며 `GeneratedPluginRegistrant` 관련 수정 시도했으나 완전 안정화는 다음 날로 이월.

## 7. 그래서 뭐가 남았나

- **메모요**: 재실행 크래시 근본 원인(UIScene) 해결, 디자인 밀도 개선, AGENTS.md 규칙 심기
- **더치페이 계산기**: 제로에서 초기 빌드 + 토스 톤 UI + App Store 업로드 시도
- **약먹자**: 다크 전환 + 핵심 입력 스크롤 휠화, 초기 커밋 `4981029` (재시작 크래시 이월)
- **고독사 방지앱**: 뼈대 + 로고 + 컨셉 피벗 + 크래시 디버깅 진행 중
- 커밋 스냅샷: `simple_memo_app@2f9f39e`, `dutch_pay_calculator@563b0d5`, `yakmukja@4981029` — 모두 원격 반영

2026.04.17 작업일지 v1.0.1

v1.0.0 이후 오전 열 시 전후부터 오후까지는 하루 종일 인프라 트러블슈팅과 설계 재정비였다. 구십오 시 오십육 분쯤 텔레그램으로 답장이 안 나가는 증상을 발견하면서 시작됐다. 내가 무언가를 답하려 해도 응답 완료 알림만 뜨고 실제 텍스트는 오지 않는 상태였다.

원인을 파고드는 일은 에이전트 세 개를 병렬로 돌려 교차 검증하는 방식으로 진행했다. 결과적으로 범인은 스케줄 잡 두 개였다. weather-dust 와 side-project-briefing 이 동일하게 오전 구 시 십오 분으로 launchd 에 걸려 있었는데, 각 잡이 claude 실행 시 채널 인자로 plugin:telegram 을 물고 올라오면서 대화형 세션과 같은 봇 토큰으로 텔레그램 Bot API 의 getUpdates 를 동시에 건드렸다. 텔레그램은 토큰당 단 하나의 폴러만 허용하기 때문에 나중에 붙은 서버가 기존 폴러를 stale 로 간주해 종료시키는 구조였고, 그 결과 대화형 세션의 MCP 가 쫓겨나고 tool 이 사라진 것이다. Stop 훅은 bash curl 로 돌아서 살아남은 반면, 답장은 MCP 경유라 침묵했다.

재발 방지로 네 개 스케줄 잡 plist 를 일 분씩 떨어진 시간대로 재배치했다. morning-reporter 는 칠 시 십 분, todo-reminder 는 칠 시 십이 분, weather-dust 는 칠 시 십사 분, side-project-briefing 은 칠 시 십육 분으로 옮기고 모든 plist 에서 채널 인자를 제거했다. 대신 스케줄 잡이 메시지를 보낼 수 있도록 ~/.claude/channels/telegram/send.sh 라는 curl 기반 헬퍼를 새로 작성했다. 이 헬퍼는 .env 의 봇 토큰을 읽어 Bot API 를 직접 호출하기 때문에 MCP 에 의존하지 않는다. 네 개 스킬 SKILL.md 도 MCP reply 호출을 send.sh 호출로 전환했다.

권한 알림 스팸을 줄이는 조정도 함께 했다. 내가 ~/.claude 경로 편집 때마다 텔레그램에 Permission Edit 과 Allowed 한 쌍이 쌓이고 있었는데, 실제 차단이 아니라 플러그인의 자동 알림 기능이었다. settings.json 의 허용 패턴에 이중 슬래시 오타가 있던 것을 단일 슬래시로 고쳐 셀프 컨피그 가드 자체를 우회하도록 바꿨다. 진행 보고 훅도 노이즈 절감 차원에서 간격을 일 분에서 이 분으로 늘리고, curl 메시지에 현재 실행 중인 툴 이름을 같이 붙이도록 보강했다. 삼 단계로 리셋 후 즉시, 오 초 뒤 스킵, 일백삼십 초 뒤 재발사를 직접 검증해 의도대로 동작하는 것을 확인했다.

오후에는 아이폰 Termius 로 Mac 에 접속이 안 된다는 문제를 잡았다. 잠시 네트워크 구조를 추적해 보니 등록된 호스트 아이피가 Tailscale CGNAT 대역이었고, 정작 Mac 쪽 Tailscale 이 중지 상태였다. 랜선을 새로 꽂으면서 Wi-Fi 에서 이더넷으로 전환되는 과정에 Tailscale 이 내려간 것이다. /Applications/Tailscale.app/Contents/MacOS/Tailscale up 한 방으로 복구했고, iPhone 측 노드도 direct 로 붙은 상태를 확인했다. 이 구조는 reference 메모리로 저장해 다음에 같은 증상이 오면 Tailscale 상태부터 보도록 했다.

마지막으로 스킬 관리 체계를 공개형으로 확장했다. 아홉 개 스킬을 ssamssae/claude-skills 저장소로 새로 올리고, daejong-page 에 Claude 스킬 카드 리스트 페이지를 추가해 네비에도 진입 버튼을 달았다. 앞으로 스킬을 만들거나 의미 있게 고치면 같은 방식으로 공개 저장소와 홈페이지 카드 리스트에 자동 반영하도록 피드백 메모리에 규칙을 박아 두었다.

할일 쪽에서는 세 건이 추가됐다. 왼손 오른손 병렬 작업 감 잡기 실습, 기존 자동화 인프라 안정화와 중복 제거, 앱 리뷰 상태 체크 알림 봇 제작이다. 이 중 마지막은 GPT 의 제안이 내 제안보다 현재 상황에 더 맞다고 판단해 반영했다. 오늘 기준 daejong-page 최신 커밋은 b573711 로 원격에 반영 완료됐고, todo 저장소 최신 커밋은 bf83b93, 새로 만든 claude-skills 저장소의 최신 커밋은 374843a 이다.

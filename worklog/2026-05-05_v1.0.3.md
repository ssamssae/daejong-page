# 2026.05.05 작업일지 v1.0.3 — 봇이 드디어 예약 버튼에 손을 댔다

> 23:00, KTX 자동예매 봇 v1의 코드가 완성됐다.
> 08:51, 아침 병렬 사이클이 시작됐을 때, 4개 앱이 동시에 손질되고 있었다.
> 00:08, 자정을 넘기기 직전 뉴스레터 Ep.7 초안이 올라갔다.

어린이날 하루가 커밋 50개 넘게 쌓이는 동안, 기록에서 빠진 조각들이 있었다. 오전 병렬 사이클에서 조용히 처리된 4개 앱, 저녁의 dutch_pay AdMob 추가, 그리고 자정에 완성된 KTX 봇. v1.0.3은 그 빈칸을 채운다.

---

## 1. 자정 직후, 뉴스레터가 먼저 움직였다

WSL이 00:08 KST에 뉴스레터 Ep.7 초안을 PR #51로 제출했다. 주제는 "2.1 Information Needed 리젝 대응기" — lottocalc가 Apple에 두 번 막히는 과정을 풀어쓴 초안. 사람들이 잠든 사이 글이 먼저 완성됐다.

---

## 2. 08:51, 아침 병렬 사이클이 열렸다

출근 직후 `parallel-cycle`을 돌렸다. Mac이 ASC URL 수정을 처리하는 동안 WSL이 4개 앱을 동시에 손질했다.

| 앱 | 작업 | SHA |
|---|---|---|
| stock_monitor | KRX 등락률 `FLUC_RT` 직접 사용으로 수정 | `0c255e7` |
| hanjul | `AiReplyBox` 공유 위젯 추출 + 삭제 다이얼로그 오류 수정 | `2e31917` |
| yakmukja | `DoseTime.periodLabel` 게터 추출 + 30일 자동 정리 | `424f7ed` |
| dutch_pay | 키패드·인원수·계산 버튼 햅틱 피드백 추가 | `9728f52` |

4개 앱을 동시에 처리하는 데 걸린 시간은 52분. v1.0.0에서 "병렬 사이클" 한 줄로만 기록됐던 것이 사실은 이 결과물들이었다.

> **배운 것** — parallel-cycle의 가치는 "WSL 3개 완료"가 아니라, 그 뒤에 커밋 메시지로 쌓이는 구체적 결과물이다.

## 3. 저녁, 두 앱이 기능을 더 달았다

**stock_monitor** — KOSPI/KOSDAQ 지수 헤더를 리스트 상단에 추가했다(`0d9a2e5`, 20:46). KRX API에서 지수값을 가져와 화면 첫 줄에 배치. 주식 앱이 비로소 시장 전체 맥락을 보여주기 시작했다.

**dutch_pay** — AdMob 배너 광고를 결과 화면 하단에 붙였다(`a3bb4de`, 21:12). `google_mobile_ads` iOS-only. 수익화의 첫 번째 배관이 연결됐다.

## 4. 코레일 봇 3단계 — 스크립트에서 봇으로

v1.0.0에서 PerimeterX에 막혀 `dead-ends`로 기록됐다. v1.0.2에서 `mitmproxy`로 iPhone과 코레일 서버 사이 트래픽을 가로채는 설정 스크립트를 완성했다. 그리고 v1.0.3에서 봇이 완성됐다.

**PR #25** (22:24) — `fix(korail-sniff): Mac 호환 IP 검출 개선`. mitmproxy 설치 경로 자동탐지 누락, `networksetup` 파싱 Mac 버전 차이 수정.

**PR #26** (22:24) — `feat(submit-app): ASC credentials 헬퍼`. 환경변수 → JSON 폴백 순서로 인증 정보를 로드하는 공통 모듈.

**KTX 자동예매 봇 v1** (23:00) — `feat(korail-sniff): KTX 자동예매 봇 v1`. 스니핑한 엔드포인트를 직접 호출해 좌석 조회 → 예약 시도. PerimeterX가 막는 웹 브라우저 경로를 우회한다.

```
booking-target.json 에 출발지·도착지·날짜·시간대 설정
→ 봇이 코레일 iOS 앱 API를 직접 호출
→ 좌석 있으면 자동 예약 시도
```

> **현 상태** — 코드 완성. 첫 실전 테스트는 iPhone 실기기에서 mitmproxy로 세션 토큰을 캡처한 뒤.

> **배운 것** — 봇탐지가 막는 건 "웹 브라우저처럼 행동하는 것"이지 "API 자체"가 아니다. 앱을 통해 같은 API를 직접 부르면 다른 게임이 된다.
